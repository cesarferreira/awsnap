## Install

```sh
yarn add makeup
```

## Usage

```
Usage

   $ makeup <command> <params>

   $ makeup sample <param>             # Uses the <PARAM>
   
 Examples

   $ makeup sample TEST                # Uses the TEST
   $ makeup sample YOLO                # Uses the YOLO
```
