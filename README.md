## Install

```sh
brew install pkg-config cairo libpng jpeg giflib imagemagick
```


```sh
npm install -g
```

## Usage

```
Usage

   $ makeup <command> <params>

 Examples
   $ makeup generate 2000 800 f7d794 tmp/ss.png tmp/output.png 1500 700    

```
